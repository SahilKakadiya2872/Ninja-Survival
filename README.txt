Overview:
This project showcases an animated virtual world created in Unity, featuring a dark forest environment with high graphics. The scene includes a wooden house inhabited by a ninja character and is surrounded by fencing. In the forest, there are various dinosaurs with random animations such as eating and roaring, enhancing the immersive experience.


Project Description:
The project aims to demonstrate the ability to animate characters within a Unity environment. The dark forest setting provides a visually rich backdrop, highlighting advanced graphical capabilities and diverse animations. The ninja character and the dinosaurs are the main focus of the animation, each performing unique actions that bring the scene to life.


Features:
High-Quality Graphics: Detailed forest environment with realistic textures and lighting.
Animated Characters: Ninja character and multiple dinosaurs with random animations.
Interactive Environment: Wooden house with fencing, adding depth to the scene.
Random Animations: Dinosaurs perform random actions like eating and roaring to simulate natural behavior.


How to run the project:
1. Install Unity Hub, Unity Editor and Visual Studio.
2. Add the project in Unity.
3. To run the game in Unity, click on the PLAY button from the main screen which is having screen index 1.
4. For testing and debugging, check the console window. 
5. To EXIT from the game, click on the PLAY button again. 


Dependencies:
Ensure you have Unity version 2020.3 or higher installed.
Install necessary packages through the Unity Package Manager (Window -> Package Manager).


How to play the game:
Use W, A, S, D Keys to control the character.
Use Spacebar to Jump.
Use C key to Crouch.
Hold Left Shift Key for faster movement.
Use mouse to control the camera.

Summary of Development/Research:

1. Distribution of Time:
Basic Knowledge of Unity (20%):
Understanding the Unity interface, including the layout and key components and familiarizing ourself with the Animator Window to manage animations.

2. Concept and Planning (10%):
To fix the concept and make a plan to go step by step with the project.

3. Asset Selecting (20%):
finding the most suitable assets from Unity Asset Store. 
 
4. Terrain and Enviroment Creation (25%)
Making a Terrain with high and low layers and making it look natural by using grass and trees.

5. Character and AI Development (25%)
Implementing animations and animation controller from the Standard Assets and making Dianosaurs animated automatically. We, also tried to make Dianosaur Controlled by AI using Nav mesh but Unfortunately, we couldn't reach there.



Challenges faced:
1. We wanted to create our own character but couldn't find suitable software for free, so had to use a character from Mixamo.
2. In the last stage, we got lagging issue while running the game.
3. We aimed to create a realistic world with high graphics, but the assets we needed were paid.
4. Many assets support different Unity versions, requiring us to download multiple versions.
5. Terrain creation was challenging.



Lessons Learned:
1. Acquired basic knowledge of Unity.
2. Learned about components, importing animations, character textures and materials, and importing assets.
3. Gained understanding of rigid bodies.
4. Learned about colliders.
5. Gained knowledge of physics materials.
6. Developed basic scripting skills using C#.
7. Learned about lighting, materials, and shaders
8. Realized that while playing a game is easy, creating one is challenging.
