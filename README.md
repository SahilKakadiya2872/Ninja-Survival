\## Project Overview



Ninja Survival is a Unity-based 3D animated scene/game prototype created to explore character animation, terrain design, asset integration, and basic gameplay controls. The environment includes a dark forest, a wooden house, fencing, animated dinosaurs, and a ninja character controlled by the player.



The main goal of the project was to understand how animated characters can be placed and controlled inside a visually rich Unity environment.



\## Features



\- Third-person ninja character movement

\- Animated dinosaur models with actions such as eating and roaring

\- Dark forest environment with realistic terrain, trees, grass, and lighting

\- Wooden house and fencing for environmental detail

\- Imported animations, textures, materials, and prefabs

\- Basic camera control and player interaction

\- Unity scene setup with multiple assets and animation controllers

\## How to Run the Project



1\. Install Unity Hub.

2\. Install Unity Editor version `2020.3` or higher.

3\. Install Visual Studio if scripting support is needed.

4\. Clone or download this repository.

5\. Open Unity Hub.

6\. Click \*\*Add Project\*\*.

7\. Select the project folder.

8\. Open the project in Unity.

9\. Open the main scene from the `Assets/Scenes` folder.

10\. Click the \*\*Play\*\* button in Unity to run the project.

11\. Click the \*\*Play\*\* button again to stop the game.

## Development Process



The project was developed step by step, starting with learning the Unity interface and Animator window. After that, the concept was planned, suitable assets were selected, and the environment was built using terrain tools, trees, grass, lighting, and imported models.



A major part of the development focused on character animation and setting up animated dinosaurs inside the environment. We also explored NavMesh-based AI movement, but it was not fully completed in the final version.


\## Time Distribution
| Task                                | Time Spent |

| ----------------------------------- | ---------- |

| Learning Unity basics               | 20%        |

| Concept and planning                | 10%        |

| Asset selection                     | 20%        |

| Terrain and environment creation    | 25%        |

| Character and animation development | 25%        |


## Challenges Faced

* Creating a custom character was difficult without suitable free tools, so Mixamo was used.
* Some high-quality assets required paid versions.
* Different assets supported different Unity versions, which caused compatibility issues.
* Terrain creation was challenging and required multiple adjustments.
* The project had performance and lag issues in the final stage.
* NavMesh-based dinosaur AI was explored but could not be fully completed.



\## Lessons Learned

* Gained basic knowledge of Unity project development.
* Learned how to import and manage assets.
* Learned how to work with animations and Animator Controllers.
* Understood character textures, materials, and models.
* Learned about Rigidbody, Colliders, and Physics Materials.
* Practiced basic C# scripting.
* Learned about lighting, shaders, and environment design.
* Understood how challenging game development can be compared to simply playing games.
