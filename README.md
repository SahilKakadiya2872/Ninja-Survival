## Challenges & Learnings

**Challenges faced:**
- Cross-version asset compatibility between Unity releases
- Performance optimisation with dense forest terrain
- Attempting NavMesh-based dinosaur AI
- Creating a custom character pipeline using free tools

**What I learned:**
- Unity Animator Controllers and state machines
- Rigidbody physics and collider configuration
- Terrain sculpting and texture painting
- Environment lighting and shader setup
- Basic C# scripting for character movement
